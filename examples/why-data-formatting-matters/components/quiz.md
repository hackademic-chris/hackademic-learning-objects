# Quiz: Check for understanding

## Question 1

Why does data formatting matter for AI adoption?

A. It makes reports look nicer  
B. It helps systems interpret information consistently  
C. It eliminates the need for governance  
D. It replaces human judgment  

**Correct answer:** B

## Question 2

Which example is easiest for automation to process?

A. “The renewal is coming up soon.”  
B. “Jane probably renews in June.”  
C. `"renewal_date": "2026-06-15"`  
D. “Ask Carlos maybe.”  

**Correct answer:** C

## Question 3

What is the main business risk in this example?

```json
{
  "status": "active",
  "approved": false
}
```

A. The data uses quotation marks  
B. The item appears active even though it has not been approved  
C. The example is too short  
D. JSON is only for developers  

**Correct answer:** B

## Question 4

What does a missing value often indicate?

```json
{
  "last_reviewed": null
}
```

A. The record may lack review evidence  
B. The system is automatically secure  
C. The data is complete  
D. The manager should ignore it  

**Correct answer:** A

## Question 5

Which is the best manager-level follow-up question?

A. “Can someone explain all of technology?”  
B. “Why is this formatted with braces?”  
C. “How will this workflow route work correctly when the required reviewer field is missing?”  
D. “Can IT fix the data?”  

**Correct answer:** C
