# Anticipation prompt

Before starting, look at these two examples.

## Example A

> Customer name: Jane Smith  
> Email jane.smith@example.com  
> Plan: Premium  
> Renewal date is sometime in June  
> Account owner maybe Carlos?  
> Notes: interested in AI features

## Example B

```json
{
  "customer_name": "Jane Smith",
  "email": "jane.smith@example.com",
  "plan": "Premium",
  "renewal_date": "2026-06-15",
  "account_owner": "Carlos Rivera",
  "interest": ["AI features"]
}
```

Answer before reading further:

1. Which example would be easier for a system to process?
2. Which example would be easier to automate?
3. Which example gives a manager more confidence?
4. What could go wrong if the organization relies on Example A?

Keep your answers. You will return to them later.
